# Altura e Largura dos Elementos

As propriedades ` height` e `width` são usadas para definir a altura e a largura de um elemento.

A altura e a largura não incluem preenchimento, bordas ou margens. Definem a altura e a largura da área dentro do preenchimento, da borda e da margem do elemento.

# Valores de altura e largura em CSS

As propriedades height e width podem ter os seguintes valores: 

* auto- Este é o padrão. O navegador calcula a altura e a largura.
* length- Define a altura ou largura em px, cm, em, etc.
* %- Define a altura ou largura em porcentagem do bloco que o contém.
* initial- Define a altura ou a largura para o seu valor padrão.
* inherit- A altura ou a largura serão herdadas do valor do elemento pai.
