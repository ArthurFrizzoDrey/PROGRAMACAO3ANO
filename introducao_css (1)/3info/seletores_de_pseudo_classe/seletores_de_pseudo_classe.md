Pseudo-classes CSS
Uma pseudoclasse CSS é uma palavra-chave que pode ser adicionada a um seletor para definir um estilo para um estado específico de um elemento.

* Alguns usos comuns para pseudoclasses:

Estilize um elemento quando o usuário mover o cursor sobre ele.
Estilize links visitados e não visitados de forma diferente.
Estilize um elemento quando ele receber o foco.
Elementos de formulário válidos/inválidos/obrigatórios/opcionais
Estilize um elemento que seja o primeiro filho de seu elemento pai.

As pseudoclasses são sempre indicadas por dois pontos (:) seguidos do nome da pseudoclasse:

* Pseudoclasses interativas
As pseudoclasses interativas aplicam estilos com base na interação do usuário :

:hover- Quando o cursor do mouse está sobre um elemento
:focus- Quando um elemento está em foco
:active- Quando um elemento está sendo ativado
:link- Links não visitados
:visited- Links visitados

* Pseudoclasses estruturais
As pseudoclasses estruturais selecionam elementos com base em sua posição na árvore do documento :

:first-child- Primeiro filho de um dos pais
:last-child- Último filho de um dos pais
:nth-child(n)- O enésimo filho de um pai
:lang()- Elementos com um idioma específico
