Seletores de atributos CSS
Os seletores de atributos CSS são usados ​​para selecionar e estilizar elementos HTML com um atributo específico, um valor de atributo ou ambos.

Os seletores de atributos estão delimitados por colchetes []. 

O CSS possui os seguintes seletores de atributos:

[attribute] - Selecione os elementos com o atributo especificado
[attribute="value"] - Selecione elementos com um atributo específico e um valor exato.
[attribute~="value"] - Selecione os elementos cujo valor de atributo contenha uma palavra específica.
[attribute|="value"] - Selecione elementos com o atributo específico, cujo valor pode ser exatamente o valor, ou começar com o valor seguido de um hífen (-)
[attribute^="value"] - Selecione os elementos cujo valor do atributo começa com um valor específico.
[attribute$="value"] - Selecione os elementos cujo valor do atributo termina com um valor específico.
[attribute*="value"] - Selecione os elementos cujo valor do atributo contém um valor específico.
