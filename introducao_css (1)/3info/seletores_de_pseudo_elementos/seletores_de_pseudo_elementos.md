Pseudo-elementos CSS
Um pseudo-elemento CSS é uma palavra-chave que pode ser adicionada a um seletor para estilizar uma parte específica de um elemento.

Alguns usos comuns para pseudo-elementos:

* Estilize a primeira letra ou a primeira linha de um elemento.
* Insira conteúdo antes ou depois de um elemento.
* Estilize os marcadores dos itens da lista.
* Estilize a parte de um elemento selecionada pelo usuário.
* Estilize a caixa de visualização atrás de uma caixa de diálogo.

Os pseudo-elementos são indicados por dois pontos duplos (::) seguidos pelo nome do pseudo-elemento:

* Pseudo-elementos de texto
Os pseudo-elementos de texto estilizam partes específicas do conteúdo do texto:

::first-line- Formate a primeira linha de texto
::first-letter- Formate a primeira letra do texto

* Pseudo-elementos de conteúdo
Os pseudo-elementos de conteúdo inserem ou estilizam o conteúdo gerado:

::before- Inserir conteúdo antes de um elemento
::after- Inserir conteúdo após um elemento
::marker- Marcadores de itens de lista de estilo
::selection- Estilizar texto selecionado pelo usuário
::backdrop- Estilo de fundo de diálogo
